🎓 CampusBot AI - Student Assistant

CampusBot AI is a Generative AI-powered student assistant chatbot built using Flask, HTML, CSS, JavaScript, and OpenRouter API.

This project supports SDG 4: Quality Education by providing students with instant educational guidance.

Features

- AI-powered chatbot
- Student assistance
- Admission guidance
- Course information
- Exam support
- Placement guidance
- Conversation memory
- Typing effect
- Modern chatbot UI

Technologies Used

- Python
- Flask
- HTML
- CSS
- JavaScript
- OpenRouter API
- Generative AI

Installation

1. Clone Repository

git clone https://github.com/yourusername/campusbot-ai.git
cd campusbot-ai

2. Install Dependencies

pip install flask
pip install flask-cors
pip install openai
Or:
pip install -r requirements.txt

Configure API Key

Open "app.py" and replace:
api_key="YOUR_OPENROUTER_API_KEY"
with your OpenRouter API key.

Run the Project

Start Flask:
py app.py
or
python app.py

You should see:
Running on http://127.0.0.1:5000
Open in Browser

Visit:
http://127.0.0.1:5000
